# Hand Gesture Mouse Control System

## Overview
This project demonstrates how hand landmarks can be used to control mouse movement and actions such as clicking, scrolling, dragging, and pausing input.

## Key Capabilities
- Cursor movement using hand position
- Left and right mouse click using finger gestures
- Scroll control using two-finger gestures
- Drag and drop functionality
- Pause and resume control for safety

## Requirements
- Python 3.10+
- Webcam
- OpenCV
- MediaPipe hand landmark model
